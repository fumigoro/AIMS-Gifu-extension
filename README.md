# AIMS-Gifu-extension

所属大学で運用されているLMS「AIMS-Gifu」ユーザー向けの拡張機能。
ファイルの一括ダウンロードができない！課題のアップロードが不便！みたいな、ちょっとした不便を解消する便利機能を詰め合わせた。
Chrome Web Storeにて公開中！
