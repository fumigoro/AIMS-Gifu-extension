# DandD-extension-for-aims

国立大学法人東海国立大学機構 岐阜大学で運用されているLMS「AIMS-Gifu」ユーザー向けの拡張機能。
この拡張機能を使用すると課題ページでファイルをドラッグアンドドロップでアップロードすることができる。
詳細は下記記事をご参照ください。
